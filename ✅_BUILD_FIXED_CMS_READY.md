## 📋 CMS USAGE

### Login:
1. Go to `https://yoursite.com/admin`
2. Enter password
3. Click "🔐 Access CMS"

### Edit Content:
1. Click "📝 Content Editor"
2. Choose tab (Homepage, About, Consultation)
3. Edit fields:
   - Left side: English (🇬🇧)
   - Right side: Arabic (🇸🇦)
4. Click "💾 Save Changes"
5. Refresh website → See changes!

### Upload Photos:
1. Click "📸 Photo Manager"
2. Choose photo slot (Hero, Headshot, Case Study)
3. Click "📤 Upload"
4. Select file
5. Preview appears
6. Done!
